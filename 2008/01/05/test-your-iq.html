测试下 你的 IQ...

2008-01-05 14:46:21 +0800

Flash :
http://61.135.153.218/vote/flash/uploadfiles/2007_02_11_03_46_20_.swf

这是一个测试 IQ 的东西，记得我在 2006 年的时候也测过一次，不过用的是另外一套题；最为神奇的是，去年找工作的时候，某个公司居然拿我做过的那套 IQ 题来做 IQ 测试，虽然做过，但是基本没啥印象了，于是乎在 30 分钟完成了大约 25% 的题，结果呢，那成功应聘，虽然最后我依然没去那家公司工作，但是依旧印象深刻，是一家算做比较前卫的公司，采用弹性工作时间，对于我这种 Programmer 来说，简直就是福音啊。

扯远了，继续扯这个 Flash 版本的 IQ 题。

我花了题目所需的所有时间来做题，在时间达到前 16 秒仓促做完了，最后结果是：
额，按其三种算法分别有三种结果：

  • 15 偏差(137满分) : 111 IQ
  • 16 偏差(140满分) : 112 IQ
  • 24 偏差(160满分) : 118 IQ

我 2006 年做的那套题的最后得分也是 118，看来还没有变笨，值得夸奖一下。

顺便，推荐一下正看这篇博客的你，也测试一下 IQ 并发来看看吧~~，记得刷新本页，从 40 分钟开始计时哟。
友情提示 : 测试只是测试而已，请勿带任何感情色彩。

最后...
恩，第一次用 Windows Live Writer 写 Blog，确实很顺手，推荐读者也用来看看哦，很舒服，还不怕丢东西，最重要的是，对于我这样一个 .Net Programmer，Live Writer 是可以根据自己需要扩展的。
Tiveone 也推荐用用 Live Writer，对于目前大多数 Blog 都支持的，只要支持 Metablog API或其他Blog 接口。
Live Writer 运行需要 .Net Framework 2.0 运行时，Vista 以下系统基本都需要自己安装，如果是 Vista 或 Win 2008 就不用操心了，因为已经捆绑安装了。
这里提供一个 Live Writer 2008 的绿色版本下载。

再次最后...，想起了

IP、IC、IQ 卡，统统告诉我密码！
