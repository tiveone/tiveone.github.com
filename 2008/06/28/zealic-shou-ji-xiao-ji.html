Zealic 手机小记

2008-06-28 14:38:04 +0800

在很久很久以前...

错了...，是在这个星期第一天的时候，我昏昏欲睡的去上班了，挤上公交车，站了会儿，忽然看见旁边的人下车了；Woo...，有位子可以打瞌睡了，我急忙凑上去坐下；只听啪的一声...

我的手机成英雄了！

原因就是我把手机上后裤袋里，并且把钥匙也放了进去。正在她们正在亲亲我我的时候，恶魔 --- 也就是我，无情的发力，让手机惨死于钥匙的剑下。

我忙不迭给钥匙解释了整整两天，终于劝住它让它不再哭泣和发飙(不然就无家可归了)，条件就是：重新给它找个老婆，要漂亮并且有内涵的；几经挑选，它选中了大家闺秀：Nokia 6120c 或 Nokia 6122c 以及 小家碧玉 HKC G801。

第三天，我和钥匙兴冲冲的冲到相亲市场去找老婆了。很可惜，华夏城主颁布了法律，禁止带 WiFi 首饰的小家碧玉入城。让钥匙心动不已的 HKC G801 就没戏了；最终钥匙在直系 Nokia 6120c 和私生子 Nokia 6122c 面前徘徊许久，终被 6122c 的朴实和温柔所打动，把 6122c 娶走做了老婆。

这彩礼可要的真不少...

给小朋友的故事到这里就结束啦，下面就是正戏：

买了 6122c 以后，我做了以下一些事:

  • 基本设置
  • 安装软件
  • 证书破解
  • 与电脑共享上网

基本设置

  1. 系统设置
    设置->手机设置->常规->个性化选择->语言->编辑语言，设定上述选项为简体中文，默认情况下为 English，这样的情况下，某些系统程序，如文件管理的改名功能，名片的改名功能会无法输入中文。

  2. 移除移动定制的资费陷阱
    由于是移动心机，移动为 6122c 定制了许多资费陷阱，一些部分是无法删除的，一些部分是可以通过程序管理删除的；我把可以删除的都删掉，不能删除的则创建一个文件夹，然后将图标丢进去。

最后排列一些功能列表，就变的很清爽干净了。

安装软件

智能机的最大好处就是可以装软件，因为 S60 本身就是一个操作系统，在此之上可以利用软件充分发挥手机的最大作用。

这里我为 6122c 安装了如下基本软件：

  1. A4 输入法 ：颇受好评的手机输入法。
  2. 手机 QQ 2008 祈福版：这个不需要介绍了吧。
  3. 飞信手机客户端：移动飞信，6122c 自带 1.02 版本，这里我删掉升级成1.2.4版本。
  4. Opera Mobile ：网络浏览器，比 6122c 自带的好很多。
  5. X-plore：可以管理系统文件，自带的文件管理看不到系统文件，是破解证书限制的必备软件。
  6. BiNPDA Security Manager(SecMan)：破解证书限制。
  7. Gnubox:：用于和电脑共享上网(需要蓝牙适配器)。
  8. 维信：让你随时随地用手机使用你喜欢的互联网应用。

证书破解

S60 第三版系统在安装应用软件时，必须提供证书，否则软件安装不上，这给我们带来了很多烦恼，当然，这个做法是好的，可以避免手机病毒、木马的侵袭；不过就像 Vista 的 UAC 一样，让人爱不起来，这里我就提供一种方法，让我们在安装软件的时候无需证书。

步骤如下：

  1. 在电脑上安装 Nokia PC Suite，并使用USB 数据线连接到手机
  2. 安装并打开 AppTRK，询问是否打开蓝牙，选择否。
  3. App TRK -> Setting ，设置 Connection 为 USB。
  4. App TRK -> Connect，确认 Status 为 Connected。
  5. 不要退出 App TRK
  6. 在电脑上安装 SecMan，一路 Next 即可。
  7. 断开数据线并删除 App TRK(不删也没问题)。
  8. 使用 x-plore，将 installserver.exe拷贝到 c:sysbin 目录下。
  9. OK

以后当需要安装软件的时候，打开 SecMan，点击选项 "Turn Plat security OFF"，然后安装软件，安装软件后再次进入，点击选项 "Turn Plat security ON"，如果 "Turn Plat security ON" 这个步骤不做的话，系统的安装性是极为低下的，极易遭到恶意软件的侵袭。

参考文章：权限与根证书完全破解!

与电脑共享上网

上网也是现在手机的重头功能之一，问题是 ISP 提供的 CMWAP，CMNET 等 GRPS 上网服务，资费贵，服务差，速度慢，并且还有很多陷阱；所以，通过电脑共享上网就成了理所当然的事情，并且这样上网没有资费忧虑，而且速度很快。

首先，我们必须准备以下东西：

  1. 蓝牙适配器(电脑)
    这是手机和电脑之间建立连接的必备工具，当然，你也可以用 USB 线通过 PC Suite 来代替，不用，那样就不是无线上网了。
    注意：购买蓝牙适配器的时候，有个原则，要买免驱动的蓝牙适配器，这是兼容性最好的蓝牙适配器。比如我的"世纪飞扬 蓝精灵 CF-WF02W" 45元，之前就买了一个兼容性极差的不兼容 XP 驱动和 WIDCOMM 驱动的杂牌蓝牙适配器，导致得到一个废品，于是才重新买了上面的蓝牙适配器。

  2. m-router(电脑)
    实现手机与电脑建立上网的网络连接的软件。

  3. Gnubox(手机)
    允许你的智能手机通过蓝牙、红外线或USB传输座,共享电脑的宽带上网。

步骤：

电脑端：

a) 电脑端安装 Widcomm 5.0.1.801(迅雷下载)，重启。
b) 电脑端安装m-Router 3.1，插入蓝牙适配器，当提示设置服务为手动时，选是。
c) 测试手机与蓝牙之间的连接，这时可以通过蓝牙传文件。

手机端：

  • 建立Bt接入点

设置->手机设置->连接->接入点，新建一个接入点，配置如下:
1. 基本配置
名称：Bt(大小写必须一致，不能为 bt 或 BT、bT)
数据承载方式：数据通话
拨号号码：321（随便）
用户名：abc
密码：xyz
其他全部默认
2. 高级设置(选项->高级设置)
代理端口号码为：6588
其他全部默认

  • 安装 Gnubox
  1. 确保蓝牙打开
  2. 安装 Gnubox
  3. 打开 Gnubox
  4. Gnubox -> Install -> Create records
  5. Gnubox -> Exit
  6. 打开 Gnubox
  7. Gnubox -> 2box Bluetooth -> Serial port
  • 测试网络
    要知道你的系统是否配置成功，通过以下步骤：Gnubox -> Debug -> Bring up IF。

如果测试成功，Gnubox 会提示 "Connection Open"，m-router 的系统托盘图标会从红色变成橙色，代表连接中，几秒后变成蓝色，表示连接成功。

现在已经完全配置好系统了，我们只需要直接启动网络软件，选择接入点为 Bt 即可。接入网络后，同样可以通过 m-router 获知连接状态。

END
